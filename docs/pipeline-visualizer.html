Information Boundary Matrix

Agent Sees code? Sees analysis? Sees notebook? Web search? Tools?
Analyst no produces yes no Read, Glob
Scientist no input yes no none
Critic no no yes yes web only
Defender no no yes yes web only
Coder only agent no no no Read, Write, Edit, Bash, Glob, Grep
Runner Python subprocess, no LLM. Executes script, captures stdout.
Why these boundaries? The plan already incorporates the analysis, so passing both to the Critic is redundant. Code is an implementation detail that only the implementer needs. The Critic and Defender debate strategy on equal footing with symmetric context. results.txt is self-compiled by the experiment script via print statements, removing the need for an LLM summarizer.

Lab notebook is a strategic journal (hypothesis, reasoning, dead ends), not a copy of model specs or detailed results. That detail lives in results.txt.
