Why these boundaries? The Scientist and Debate both see the analysis JSON and prediction history to ground their reasoning in evidence. Code is an implementation detail that only the Coder needs. The Debate uses 4 specialized personas (Methodologist, Trajectory Critic, Falsification Expert, Evidence Auditor) to cover different failure modes. After debate, the Scientist revision incorporates the structured concern ledger; the Coder gets only the revised plan.

Prediction history tracks testable predictions as a tree (via follows_from links). The Scientist creates predictions, the experiment script evaluates them in code, and the Analyst transcribes outcomes. This replaces the earlier success_criteria system.

Lab notebook is a strategic journal (hypothesis, reasoning, dead ends), not a copy of approach specs or detailed results. That detail lives in results.txt.

Web search is available to the Scientist (plan), Debate (both sides), and Scientist (revision). This grounds hypotheses and critiques in real science. The Analyst and Coder do not have web access.
